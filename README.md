## 🧑‍💻 About Me

There was a point where "making progress" just meant getting the code to run.  
I'd skim docs, copy snippets, and move on — until I couldn't debug anything without guessing.

That's when I shifted: instead of just implementing, I started asking *what's actually happening here?*  
Now I focus on understanding request flow, dependency injection, and system architecture — not just making frameworks do their thing.

- 🎓 Systems Engineering · 7th semester · Universidad de Antioquia  
- 🏫 Teaching Assistant — Programming Techniques (OOP, Java, Git, Scrum)  
- 🔒 Working with Spring Security, JWT, and layered architecture in real team projects  
- 📐 Interested in SOLID principles, Clean Code, and design patterns  
- 🌱 Currently deepening knowledge in system design and backend architecture  
- 🚀 Open to junior backend opportunities
